Arquitetura E5R

A Arquitetura E5R é basicamente a forma como eu Erlimar costumo organizar meus projetos .NET, e também reúne uma "meia dúzia" de utilitários, extensões, genéricos e outras classes úteis. Então podemos dizer que E5R.Architecture é uma especificação + alguns utilitários que ajudam você a implementar uma arquitetura de software .NET.

Basicamente, este projeto começou com uma série de utilitários e genéricos que uso para abstrair o acesso ao banco de dados nas aplicações .NET que tenho trabalhado, e mais especificamente usando o EntityFramework, mas também incluía algumas outras "coisinhas" para auxiliar na forma como implemento essas aplicações seguindo os princípios de DDD.

Além das idéias de organização em si (que iremos discutir nos próximos tutoriais), estão disponíveis os seguintes componentes que podem ser obtidos via NuGet.
