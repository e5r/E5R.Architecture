Remarks

If there is an ITransformer<TFrom, TTo> from TFrom to TTo it will be used, in another case a new instance of TTo will be created and the property values will be copied from TFrom to TTo using CopyPropertyValuesTo<TFrom, TTo>(TFrom, TTo).
