TryFromTag<TEnum>(String, String, out TEnum)

Get a enum value from a tag value. A return value indicates whether the get was successful.

Declaration
public static bool TryFromTag<TEnum>(string tagKey, string tagValue, out TEnum result)
    where TEnum : Enum
Parameters
Type Name Description
System.String tagKey

The tag key
System.String tagValue

The tag value
TEnum result

When this method returns, it contains the enum value annotated with the tag key tagKey that has the tag value tagValue, if the get was successful, or the default value of the enum if the get failed. The get will fail if there is no enum value annotated with the given tag. This parameter is passed uninitialized; any value originally given in result will be replaced.
Returns
Type Description
System.Boolean

true if the tag was obtained; otherwise, false.
